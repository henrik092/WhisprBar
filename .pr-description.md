## 🔴 Critical Bugs Fixed (P0)

### 1. VAD Auto-Stop Queue Race Condition
**File:** `audio.py`
**Problem:** VAD monitor read from queue with `get_nowait()` then put items back, causing duplicates/loss
**Solution:** Separate `VAD_MONITOR_QUEUE` for non-destructive monitoring
**Impact:** ✅ No more audio chunk corruption

### 2. Overlay Window Race Condition
**File:** `ui.py`
**Problem:** `_overlay_window` checked without lock, race between threads
**Solution:** Added `_overlay_window_lock` with double-check pattern
**Impact:** ✅ No more multiple overlay windows or memory leaks

### 3. Data Loss in stop_recording()
**File:** `audio.py`
**Problem:** Complex queue drain with nested timeouts, last audio could be lost
**Solution:** Simplified logic - stop stream first, then single drain loop
**Impact:** ✅ No more missing audio at recording end
